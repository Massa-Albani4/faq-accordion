# Faq Accordion  

## Overview  
A web app that tracks real-time stock prices using WebSockets and visualizes trends with D3.js.  

## Features  
- Live stock updates  
- Interactive charts  
- Dark mode support  

## Tech Stack  
- React, TailwindCSS  
- WebSockets for real-time data  
- D3.js for data visualization
